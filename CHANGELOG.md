# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-04-25

### Added
- `analyzeProse` combined analysis function in `index.ts`.
- Enhanced `README.md` with "Quick Start" and detailed feature breakdown.
- Dependency count badge in `README.md`.

## [1.0.0] - 2026-04-25

### Added
- Initial release of `@veldica/prose-analyzer`.
- Lexical diversity metrics (TTR, MATTR).
- Repetition detection and stopword filtering.
- Narrative texture analysis (Dialogue Ratio, Scene Density, Exposition Density).
- Sensory and Abstract language signal detection.
- Sub-module support for modular imports.
